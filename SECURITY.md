# Security Policy

## Reporting a Vulnerability

For now, please report potential vulnerabilities by opening a private issue or contacting the maintainers directly.

Security contact:

- Email: `anton.f@synqra.tech`

Include:

- affected component/file,
- reproduction steps,
- expected vs actual behavior,
- potential impact.

Please avoid publishing exploit details publicly before maintainers acknowledge and triage the report.

## Scope Notes

`omega-walls` OSS is a reference runtime layer. It does not include enterprise control plane or managed operations.
