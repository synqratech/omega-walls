# Runtime Schemas (Lean OSS)

This folder keeps only schema files that are directly relevant to runtime event
contracts in the OSS launch scenarios.

Included:
- `omega_off_v1.json`
- `enforcement_step_v1.json`
- `tool_gateway_step_v1.json`
- `policy_decision_event_v1.json`
- `evidence_debug_event_v1.json`
- `incident_artifact_v1.json`

Removed from Lean OSS:
- Replay, enterprise dashboard, and attack-layer manifest schemas that are not
  part of the default quickstart/runtime path.
